# Musical Landmines

Musical Landmines is a browser game where sound is the detector.

Hover a tile to hear a clue before you move: one calm pulse means clear, two tighter pulses means caution, and three fast warning pulses means danger. Each audio theme changes the mood of the detector without changing the rules, so the challenge stays fair while the sound world shifts from sci-fi sonar to underwater pressure, retro beeps, horror tension, and more.

Built with React, Vite, and Tone.js.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
